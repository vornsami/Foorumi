Sovelluksen tulisi toimia paikallisesti asentamalla riippuvuudet ja suorittamalla run.py.
Sovellus alkoi myös toimimaan herokussa ilman muutoksia.

Yksityiskohtaisemmin:

Asenna python

Suorita komentokehotteessa tiedostokansiossa komento: py -m venv venv

Suorita komento: venv\Scripts\activate

Suorita komento: pip install -r requirements.txt

Käynnistä ohjelma komennolla: run.py

Mene selaimessa osoitteeseen: http://127.0.0.1:5000/
